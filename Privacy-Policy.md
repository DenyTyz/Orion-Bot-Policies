# Privacy Policy for Orion Bot
**Last Updated:** May 2026

## 1. Data We Collect
When you invite and use Orion in your Discord server, we may collect and store the following data:
* **Discord User IDs:** Used for the Premium system, dev recognition, and message trackers.
* **Discord Guild (Server) IDs:** Used to store server-specific settings (like custom prefixes) and message trackers.
* **Message Activity:** The bot reads messages to execute commands and count message activity for tracker features, but **we do not store the content** of the messages, only the numerical count.

## 2. How We Use Your Data
The collected data is solely used to provide the bot's core functionalities, such as:
* Executing commands efficiently.
* Saving server preferences (e.g., custom prefix).
* Managing user Premium subscriptions and server boosts.

## 3. Data Sharing
We **do not** sell, rent, or share user or server data with any third parties. Data is kept private and secure.

## 4. Data Deletion
If you wish to have your data removed from our database, you can simply kick the bot from your server or contact the developer.
